* Gustavo Oliva

## Descripción:
PET Center es una plataforma orientada a conectar dueños de mascotas con proveedores de servicios como veterinarias, peluquerías, paseadores, cuidadores y tiendas especializadas.
El proyecto fue gestionado utilizando metodología **Scrumban**, combinando prácticas de Scrum y Kanban para la planificación, seguimiento y control de las actividades.

## Gestión del Proyecto
Durante el desarrollo se implementaron las siguientes herramientas de gestión:
* Product Backlog
* Historias de Usuario
* Sprints
* GitHub Issues
* GitHub Milestones
* Tablero de seguimiento en Trello
* Control de versiones mediante Git

## Milestones
Se definieron los siguientes hitos principales:

### M1 — Base del Proyecto
Funcionalidades iniciales de registro, autenticación y búsqueda de servicios.
### M2 — Núcleo Funcional
Gestión de mascotas, reservas, disponibilidad y valoraciones.
### M3 — Plataforma Completa
Administración de usuarios y proveedores, integración, pruebas y requerimientos no funcionales.
